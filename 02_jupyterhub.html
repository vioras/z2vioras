VIORAS JupyterHub#

ここでは、WebブラウザからVIORASのJupyterHubに接続し、 VIORASがサービス提供しているChatGPT (Jupyter AI Chat) を使ってPythonのプログラミングを自習する手順を説明します。

Note

“Jupiter” のミス・スペルではありません Jupyterの名前は、主にサポートしている3つのプログラミング言語、Julia、Python、そしてRから取られています。

VIORAS JupyterHubの主な特徴#

  1. 最新のPythonとJuliaによる対話的なプログラミング (今のところRはサポートしていません)

  2. Language Server Protocolによるプログラミング支援

  3. Jupyter AIによるChatGPTとの対話機能

  4. MyST Markdownによる書式整形

  5. Azure Storageによるオブジェクト・ストレージ

VIORAS JupyterHubに接続する#

Webブラウザから https://ecsosaka-jh2300.japaneast.cloudapp.azure.com/ にアクセスするとVIORAS JupyterHubに接続し、 認証を促す “Sign in with Azure AD” というボタンが表示されます。 ボタンを押すと、Microsoftのサインイン画面に遷移するので、 OUMailアカウントで多要素認証を行ってください。

  1. OUMailアカウントのユーザ名とパスワードを入力すると2桁の数字コードが表示される

  2. 連携しているスマートフォンのAuthenticatorアプリがコード入力を促す画面に遷移しているので、表示された2桁の数字コードを入力する

実行するサーバを選択する#

2023年度秋冬学期から、システム更新に伴って、 接続時に実行するサーバを選択する必要があります。 現在、提供されているのは次の2種類のサーバになります。

  1. default

    • Pythonのみ使える

  2. w/julia

    • PythonとJuliaが使える

授業でいずれのサーバを利用するかは教員の指示に従って下さい。 特に指示がない場合 (Juliaを使わない場合) は、そのままStartボタンを押してください。

VIORAS JupyterHubを終了する#

JupyterHubを終了するには、FileメニューからLog Outを選択してください。

この操作は、Jupyterが動作しているサーバを停止していませんので、直後であればセッションに再接続して作業を再開することができます。 ただし、無操作状態が続くとサーバは自動停止します (一時間くらい)。それ以降に再接続すると改めてサーバが起動し、新しいセッションが開始されます。

ファイルブラウザをJupyter AI Chatに切り替える#

VIORASは、標準的なJupyterLabに対して多くの機能拡張を行っています。 最も最新の機能拡張はJupyter AIで、VIORASではユーザ登録をしなくてもChatGPT (GPT-4/32,200コンテキスト) を使うことができます。

標準では「ファイルブラウザ」が有効になっています。

左サイドバーの縦アイコン列からチャットマークを選択すると、左サイドバーがJupyter AI Chatに切り替わり、ChatGPTとの対話が可能になります。

ChatGPTを家庭教師にして自習する#

例えば次のように入力してJupyterでのプログラミングを初めてください。 これらの入力をプロンプトとよび、ChatGPTからの答えをレスポンスといいます。

  1. 「JupyterHubにログインしました。JupyterLabが起動しているそうです。」

  2. 「今からPythonで初歩的なプログラミングをしてみたいと思います。操作手順をステップバイステップで教えてください。」

    • 「ランチャーとは何ですか？」

    • 「ファイルブラウザーとは何ですか？」

  3. 「ノートブックに名前を付けて保存して、ダウンロードする手順を教えてください。」

Note

“Jupyternaut”は、 JupyterLabから利用できる会話型のアシスタントの名前です。 ここで紹介した方法で問い合わせをすることができますが、会話は保存されないことに注意してください。

Note

VIORAS JupyterHubはOUMailアカウントで認証していますが、 Jupyterシステム上は”jovyan”という匿名のユーザ名に対応づけられています。 “Jovian”（木星の、木星に関する、という意味）から来ており、 Jupyterプロジェクトの天文学へのオマージュを反映しています。 スペルが違いますが、利用中に意識することはないでしょう。
